# Expense Tracker (Python)

## Description
This is a simple console-based expense tracker built using Python.
It allows users to add expenses, view them, calculate total expenses,
and find the highest expense.

## Features
- Add multiple expenses
- View all expenses
- Calculate total spending
- Identify highest expense category
- Input validation using try-except

## Technologies Used
- Python

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python expense_tracker.py
3. Choose options from the menu

## Learning Outcomes
- Functions in Python
- Lists and dictionaries
- Exception handling
- Menu-driven programs
